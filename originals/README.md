# Inbox de prints crus

Solte aqui os screenshots **sem tratamento**.

Fluxo:
1. Você coloca os prints crus nesta pasta.
2. O Claude anota (saída em `/images/anotadas/`), mede com a grade e atualiza a página.
3. Depois de anotado, o cru é **apagado** desta pasta — a versão anotada é a que fica.

Assim o inbox fica sempre limpo.
